提交范围：Monorepo、前端/后端启动骨架、Docker 基础、README、演示脚本、验收清单。

提交范围：知识库、文档上传、文档解析、文本切片、检索、RAG 问答、引用来源。

## Week 4：Prompt / 日志 / 成本

提交范围：Prompt 模板、变量渲染、调用日志、成本估算、Dashboard 数据聚合。

## Week 5：Workflow 编排

提交范围：节点协议、执行器、节点运行记录、前端运行面板、后续 React Flow 接入点。

## Week 6：Eval / 部署 / 面试包装
