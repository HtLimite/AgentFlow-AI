# 面试讲解稿

## 一句话

AgentFlow-AI 是一个企业级 AI Agent 工作流与知识库平台，覆盖模型接入、RAG、工具调用、工作流、Prompt 管理、调用日志、成本统计、评测和 Docker 部署。

## 讲解顺序

1. 为什么做这个项目
2. 解决什么企业问题
3. 整体架构是什么
4. RAG 怎么实现
5. Agent 怎么调用工具
6. 工作流怎么执行
7. Prompt 怎么管理
8. 日志和成本怎么统计
9. 评测怎么做
10. 部署怎么做

## 核心表达

企业落地 AI 不只是接一个聊天接口，真正的问题是知识怎么接入、工具怎么调用、流程怎么控制、结果怎么评估、成本怎么观测、权限怎么隔离。所以我把 AgentFlow-AI 设计成完整的企业级 AI 应用平台。
