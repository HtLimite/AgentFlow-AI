# Roadmap

## V1.5：当前版本

目标：可启动、可演示、可放 GitHub。

- Monorepo 工程结构
- 前端管理台
- FastAPI 后端
- 模型供应商 CRUD
- 敏感字段加密保存
- Chat Playground
- 知识库上传、切片、RAG 问答
- Agent 工具调用演示
- Prompt 测试与版本演示
- Workflow 节点协议与执行器
- Eval 评测演示
- Docker Compose
- CI
- README 与面试文档

## V2：主项目增强

目标：从演示闭环升级到真实工程闭环。

- Alembic 数据库迁移
- 知识库、文档、切片持久化
- MinIO 文件存储
- 真实 Embedding 服务
- pgvector 向量检索
- OpenAI-Compatible Chat Completion
- SSE Streaming 真流式输出
- LLM 调用日志持久化
- Token 成本统计持久化
- Agent Tool Registry 标准协议

## V3：高竞争力版本

目标：形成 AI 平台感。

- React Flow 工作流画布
- LLM 节点配置面板
- Knowledge 节点配置面板
- Condition 分支节点
- HTTP 节点配置面板
- 工作流运行日志
- Prompt Diff 与回滚
- LLM-as-Judge 自动评测
- 模型效果对比
- Dashboard 完整图表

## V4：企业化能力

目标：接近商业平台。

- 多租户
- RBAC 权限
- 数据脱敏
- 审计日志
- SQL 工具更严格的安全策略
- 在线 Demo
- 项目官网
- 演示视频
